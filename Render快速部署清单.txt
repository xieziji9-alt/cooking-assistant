📝 Render部署步骤（5分钟）
═══════════════════════════════════════════════════════

第1步：登录Render
□ 访问：https://dashboard.render.com
□ 点击 "Sign in with GitHub"
□ 授权Render访问GitHub

第2步：创建Static Site
□ 点击 "New +" 按钮
□ 选择 "Static Site"

第3步：连接仓库
□ 找到：xieziji9-alt/cooking-assistant
□ 点击 "Connect"

第4步：配置设置
□ Name: cooking-assistant
□ Branch: main
□ Root Directory: website  ⚠️ 重要！
□ Build Command: （留空）
□ Publish Directory: .

第5步：部署
□ 选择 Free 计划
□ 点击 "Create Static Site"
□ 等待1-2分钟

🆘 常见问题
═══════════════════════════════════════════════════════

问题1：找不到仓库
→ 点击 "Configure account"
→ 授权Render访问您的仓库

问题2：部署失败，显示404
→ 检查 Root Directory 是否填写了 "website"
→ 检查 Publish Directory 是否填写了 "."

问题3：网站显示空白
→ 检查GitHub仓库是否有文件
→ 检查文件是否在website文件夹里

问题4：还没上传到GitHub
→ 先完成GitHub上传
→ 或者使用Netlify Drop（无需GitHub）
