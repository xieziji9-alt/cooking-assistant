## 🚀 快速启动（3种方式）

### 方式1️⃣：双击启动脚本（最简单）
1. 进入 `website` 目录
2. 双击 `启动网站.bat`

1. 进入 `website` 目录

### 方式3️⃣：使用本地服务器（推荐）

## 🎯 核心功能实现

### 智能推荐算法
```
1. 遍历所有菜谱
2. 计算食材匹配数
3. 计算厨具匹配数
4. 计算总匹配分数
5. 按总分、食材分、匹配比例排序
6. 返回推荐结果
```

### 随机推荐算法
```
1. 选择推荐数量
2. 使用Set确保无重复
3. 随机选择菜谱
4. 返回推荐结果
```

### 自动生成菜谱
```
1. 为每种食材生成多道菜
2. 确保每种食材至少3道菜
3. 总菜谱数达到320+
4. 自动匹配B站视频链接
```

## 📝 常见问题

### Q1: 网站打不开？
**A:** 确保使用现代浏览器，建议使用Chrome或Edge。

### Q2: 视频链接打不开？
**A:** 视频链接跳转到B站，需要网络连接。链接已自动复制到剪贴板，可以手动粘贴打开。

### Q3: 推荐结果为空？
**A:** 尝试选择更常见的食材，或者减少厨具限制。

### Q4: 可以离线使用吗？
**A:** 可以！除了B站视频链接需要网络，其他功能都可以离线使用。

### Q5: 可以添加自己的菜谱吗？
**A:** 可以！编辑 `app.js` 文件，在 `recipes` 数组中添加新菜谱即可。
