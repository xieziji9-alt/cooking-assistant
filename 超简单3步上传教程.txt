🚀 超简单3步上传教程
═══════════════════════════════════════════════════════

您的GitHub仓库：
https://github.com/xieziji9-alt/cooking-assistant

═══════════════════════════════════════════════════════

第1步：上传文件（2分钟）
═══════════════════════════════════════════════════════

1. 打开您的仓库页面（已在浏览器打开）
   https://github.com/xieziji9-alt/cooking-assistant

2. 看到页面中间有一个虚线框，写着：
   "Quick setup — if you've done this kind of thing before"
   
   在下面找到这句话：
   "...or create a new repository on the command line"
   
   再往下，找到蓝色链接：
   "uploading an existing file"
   
   点击这个蓝色链接！

3. 现在会看到一个上传页面，写着：
   "Drag files here to add them to your repository"

4. 打开文件夹：
   d:\code\idea\ideaPrograms\zuocai\website\

5. 选中这5个文件（按住Ctrl点击）：
   ✅ index.html
   ✅ styles.css
   ✅ script.js
   ✅ app.js
   ✅ autoRecipes.js

6. 拖拽这5个文件到GitHub页面的虚线框里

7. 等待文件上传完成（会显示绿色的勾）

8. 滚动到页面底部，看到：
   "Commit changes"
   
   在 "Commit message" 输入框填写：
   Add cooking assistant website

9. 点击绿色按钮 "Commit changes"

✅ 第1步完成！

═══════════════════════════════════════════════════════

第2步：启用GitHub Pages（1分钟）
═══════════════════════════════════════════════════════

1. 在仓库页面，点击顶部的 "Settings" 标签
   （在 Code、Issues、Pull requests 那一行的最右边）

2. 在左侧菜单，向下滚动，找到 "Pages"
   （在 "Code and automation" 部分）
   
   点击 "Pages"

3. 在 "Build and deployment" 部分：
   
   Source: 选择 "Deploy from a branch"
   
   Branch: 
   - 第一个下拉框选择 "main"
   - 第二个下拉框选择 "/ (root)"
   
   点击 "Save" 按钮

4. 页面会刷新，顶部会出现蓝色提示框：
   "GitHub Pages source saved."

✅ 第2步完成！

═══════════════════════════════════════════════════════

第3步：访问您的网站（1分钟）
═══════════════════════════════════════════════════════

1. 等待1-2分钟（GitHub正在部署您的网站）

2. 刷新Pages设置页面（按F5）

3. 页面顶部会显示绿色提示框：
   "Your site is live at https://xieziji9-alt.github.io/cooking-assistant/"

4. 点击这个链接，或者在新标签页打开：
   https://xieziji9-alt.github.io/cooking-assistant/

5. 您的做菜小助手网站就上线了！🎉

✅ 第3步完成！

⚠️ 如果遇到问题
═══════════════════════════════════════════════════════

问题1：找不到 "uploading an existing file"
→ 刷新页面
→ 或者点击 "Add file" → "Upload files"

问题2：网站显示404
→ 等待3-5分钟
→ 清除浏览器缓存（Ctrl+F5）
→ 检查仓库是否Public

问题3：Pages设置里没有Save按钮
→ 检查是否选择了Branch
→ 确保仓库是Public
