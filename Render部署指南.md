# 🚀 Render.com 部署指南

## ✅ Render.com 的优势

相比GitHub Pages，Render更简单：

- ✅ **无需Git命令** - 直接拖拽文件夹
- ✅ **无需Token** - 只需登录账号
- ✅ **自动部署** - 上传即部署
- ✅ **永久免费** - 静态网站完全免费
- ✅ **自定义域名** - 支持免费域名

---

## 📝 部署步骤

### 方法1：通过GitHub连接（推荐）

如果您的GitHub仓库已经有代码：

1. 在Render Dashboard点击 "New +" → "Static Site"
2. 连接您的GitHub账号
3. 选择仓库：`cooking-assistant`
4. 配置：
   - Name: `cooking-assistant`
   - Build Command: 留空
   - Publish Directory: `.`
5. 点击 "Create Static Site"

### 方法2：直接上传文件（更简单）

如果不想用GitHub：

1. 创建一个新的Static Site
2. 选择 "Deploy from Git" 或 "Manual Deploy"
3. 上传您的文件
